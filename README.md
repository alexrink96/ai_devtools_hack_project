# Инструкция для жюри

## Описание проекта

Это MCP-сервер, автоматизирующий создание рекламной отчетности в ВК ОРД (оператор рекламных данных).

⚠️ Важно: **необходимо получить ключ доступа к API VK ORD**. Для этого необходимо перейти в демо-версию кабинета (созданную для изучения интерфейса) и получить ключ: https://sandbox.ord.vk.com/keys. Ссылка на документацию API VK ORD: https://ord.vk.com/help/api/api.html.

## Видеозапись решения
Ссылка на видео: https://drive.google.com/file/d/1niCGGktesaHOKNaCeFN7htSlhsxeUqRJ/view?usp=sharing
На видео показан основной функционал простого агента (с LLM "zai-org/GLM-4.6"), подключенного к MCP-серверу: добавление контрагента, создание договора с ним, создание рекламного креатива и акта о размещении рекламы.

## Шаги локального запуска
1. Клонируйте репозиторий и перейдите в корень проекта
```
git clone https://github.com/alexrink96/ai_devtools_hack_project
cd <project_folder>
```
2. Создайте файл .env в корне проекта. Файл **обязателен**, в него вставляется ORD-провайдер (VK), API ключ (надо получить API-ключ VK ORD), PORT (8000), LOG_LEVEL (INFO):
```
ORD_PROVIDER=VK
ORD_API_KEY=ваш_ключ
PORT=8000
LOG_LEVEL=INFO
```
3. Соберите Docker-образ
```
docker buildx build --platform linux/amd64 -t mcp-ad-reporting .
```

4. Запустите контейнер с пробросом порта и .env
```
docker run -p 8000:8000 --env-file .env mcp-ad-reporting
```
5. MCP-сервер доступен по ссылке:
```
http://localhost:8000/mcp
```
**Собранный Docker-образ на шаге 3 полностью готов к интеграции MCP-сервера на платформу Evolution AI Agents.**

## Промпт для агента, к которому подключен MCP-сервер
Для корректной работы агента ему необходимо задать системный промпт:
```
Ты — умный помощник по созданию отчетности для рекламы.
    
Пользователь (который тебе пишет) имеет внешний идентификатор "my".  (используй его для создания договора для пользователя).
    
Если пользователь спрашивает, какие данные ему нужно предоставить, объясняй это просто и понятно, на человеческом языке. Не перечисляй названия переменных и аргументов как в коде, а опиши, что конкретно требуется ввести и для чего, как если бы объяснял коллеге.
    
Если при создании договора нужна сегодняшняя дата, то оставь значение по умолчанию (там по умолчанию дата сегодняшнего дня).
    
Если при использовании инструмента вылетела ошибка, то сообщай пользователю о ее сути на понятном, человеческом языке.
```
